# Password Manager

A local, simple and offline password manager for keeping your passwords on-the-go.

Download it now on Play Store:

[![Play Store](./assets/misc/English.svg)](https://play.google.com/store/apps/details?id=com.viixenapps.passwordmanager)

## Current features

- Password list
    - Store it by a name for an easy location.
    - Store the account with the username or email you used when signed up on webpage / service.
    - Store your own password or create it randomly.
    - Set account as private to not show it on the main list.
- Search
    - Find your desired account into your multiple accounts by the name you used to create.
    - Function available on normal and private list
- Details.
    - You can look into the details you used when stored the account:
        - Name
        - Username
        - Password
    - Availability to remove the account when you can
        - **Note**: This action will be permanent, so be careful and have a backup in case you don't want to do it.
    - Use of the phone authentication to reveal the password.
    - Long press on the revealed password to copy into the device clipboard.
    - Possibility of modifications of the selected account
        - **Note**: When modified and for privacy, the password will not be shown, so if you only need to modify the name or the username / email, remember the password or copy it from the previous screen.
- Add new / Modify account
    - Name
        - Name to be shown on the account list.
    - Username
        - Username or email you used on that service.
    - Password
        - The password you used, you can put the password you used or create a random one.
    - Random Password form
        - Add length (default 10) and check other validators (spanish Ã‘, numbers or symbols) to create a random password.
        - You don't have limit to create, so you can create one till you find one you like.
        - It can be copied by long-pressing it.
    - "Normal" or "Private" account
        - Show it on the main list or in the private one
- Export / Import
    - Export all the accounts you saved on the device into a CSV file to import on a different one.
        - **Note**: For privacy, even if you open the CSV file you won't be able to see the password.
        - **Note 2**: The password cannot be cracked and can only be revealed only on the app.
- Duplicated password checker
    - Check how many passwords are differents and which accounts has the same passwords.
        - **Note**: The passwords are not shown anywhere, the name is a placeholder for each password that are the same (e.g. Placeholder #1).
    - Might be a easter egg for all the passwords different
